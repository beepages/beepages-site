BEEPAGES ABOUT COMMUNITY TUTORIALS DOWNLOAD

Download BeePages

Download RAR (v0.6.1)

Prerequisites

Once you have BeePages extracted to your site folder, check that you have NodeJS globally installed

node -v

and

npm -v

If you do not have NodeJS installed yet, install it.

Installing BeePages

After you download the RAR file, extract it to a folder for BeePages to run from. Then run (in cmd)

npm update

in your folder to install the Node modules needed.

Starting your site

To start the BeePages instance, execute init_local.bat or run

npm start

Accessing your site

Your site can be found at http://localhost:8080/and you can find the admin tools at http://localhost:8080/admin after initializing BeePages.
